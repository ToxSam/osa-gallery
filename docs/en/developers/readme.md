# For Developers

Welcome to the developer resources section! Here you'll find everything you need to integrate Open Source Avatars into your projects, access our APIs, and contribute to the ecosystem.

## Overview

Open Source Avatars provides a comprehensive database of CC0 and open source avatars that you can integrate into your applications, games, VR experiences, and more. All resources are freely available and designed to be developer-friendly.

## What's Available

Our developer resources include:

- **Avatar Database**: Complete JSON database with metadata, file URLs, and project information
- **API Access**: Programmatic access to avatar data and collections
- **Source Code**: Open source codebase for the gallery website
- **Documentation**: Detailed guides and examples for integration

## Getting Started

To get started, explore our avatar database structure and learn how to access avatar data programmatically. All avatars are stored on decentralized storage (Arweave, IPFS) ensuring permanent availability.
