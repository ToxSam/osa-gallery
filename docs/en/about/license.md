# License Guide

> **License matters!** CC0 avatars can be used freely without credit. CC-BY avatars require attribution to the original creator. Always check the collection details.

## CC0 (Public Domain)

CC0 means the work is in the public domain. You can:

- Use commercially without restrictions
- Modify and remix freely
- No attribution required
- Use in any project, including commercial

## CC-BY (Attribution Required)

CC-BY allows free use but requires attribution:

- Use commercially
- Modify and remix
- Must credit the original creator
- Link to original work when possible
