# 開発者向け

開発者向けリソースセクションへようこそ！ここでは、オープンソースアバターをプロジェクトに統合し、APIにアクセスし、エコシステムに貢献するために必要なすべての情報を見つけることができます。

## 概要

オープンソースアバターは、アプリケーション、ゲーム、VR体験などに統合できるCC0およびオープンソースアバターの包括的なデータベースを提供します。すべてのリソースは無料で利用でき、開発者フレンドリーに設計されています。

## 利用可能なもの

開発者向けリソースには以下が含まれます：

- **アバターデータベース**：メタデータ、ファイルURL、プロジェクト情報を含む完全なJSONデータベース
- **APIアクセス**：アバターデータとコレクションへのプログラムアクセス
- **ソースコード**：ギャラリーウェブサイトのオープンソースコードベース
- **ドキュメント**：統合のための詳細なガイドと例

## はじめに

始めるには、アバターデータベース構造を探索し、プログラムでアバターデータにアクセスする方法を学びます。すべてのアバターは分散型ストレージ（Arweave、IPFS）に保存されており、永続的な可用性を保証しています。

## リソース

- [アバターデータベース](/ja/resources/developers/database)：JSON構造とGitHubリポジトリ
- [ウェブサイトソースコード](/ja/resources/developers/website)：ギャラリーウェブサイトのGitHubリポジトリ
