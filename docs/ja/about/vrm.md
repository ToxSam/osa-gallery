# VRM形式について

VRMはゲーム、VR、仮想世界向けに設計された3Dアバターファイル形式です。

## VRMを使用する理由：

- 複数のプラットフォームで動作（Unity、ウェブブラウザー、VRアプリ）
- オープンスタンダード - 1つの企業にロックされない
- 簡単なアニメーションのための組み込みリギングを含む
- 表情とカスタマイズをサポート

## 互換性：

- VRChat、VSeeFace、VTuberアプリ
- Unity（UniVRMを使用）
- ウェブブラウザー（three-vrmを使用）
- Unreal Engine（VRM4Uを使用）

[VRMについて詳しく見る →](https://vrm.dev/ja/)
