# ライセンスガイド

> **ライセンスは重要です！** CC0アバターは帰属表示なしで自由に使用できます。CC-BYアバターは元の作成者への帰属表示が必要です。常にコレクションの詳細を確認してください。

## CC0（パブリックドメイン）

CC0は、作品がパブリックドメインにあることを意味します。以下が可能です：

- 制限なく商用利用
- 自由に修正およびリミックス
- 帰属表示不要
- 商用を含むあらゆるプロジェクトで使用

## CC-BY（帰属表示が必要）

CC-BYは自由な使用を許可しますが、帰属表示が必要です：

- 商用利用可能
- 修正およびリミックス可能
- 元の作成者へのクレジットが必要
- 可能な限り元の作品へのリンクを提供
