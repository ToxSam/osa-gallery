# について

オープンソースアバターの背後にある技術と理念について学びます。

## 私たちのミッション

オープンソースアバターは、誰もが無料でアクセス可能な高品質な3Dアバターを提供することに専念しています。私たちはオープンスタンダード、分散型ストレージ、コミュニティ主導のコンテンツの力を信じています。

## 使用している技術

アバターがアクセス可能で、永続的で、プラットフォーム間で互換性があることを保証するために、最新の技術を活用しています：

- **VRM形式**：複数のプラットフォームで動作する汎用3Dアバター形式
- **分散型ストレージ**：ArweaveとIPFSでの永続的なストレージにより、アバターが常に利用可能であることを保証
- **オープンスタンダード**：独自のソリューションではなく、オープンな形式と標準を使用

## 私たちの理念

私たちはオープンソースの原則にコミットしています。ほとんどのアバターはCC0（パブリックドメイン）ライセンスを使用しており、帰属表示なしで自由に使用できます。一部のコレクションはCC-BYを使用しており、帰属表示が必要ですが、商用利用も可能です。
