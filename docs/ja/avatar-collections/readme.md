# アバターコレクション

アバターコレクションセクションへようこそ！このサイトは、異なるライセンスとソースを持つ複数のアバターコレクションをホストしており、すべてプロジェクトで無料でアクセス可能になるように設計されています。

## 概要

オープンソースアバターは、ToxSamによって作成されたオリジナルコレクションと、才能あるクリエイターからのコミュニティコレクションの両方を特徴としています。すべてのコレクションは、プラットフォーム間での品質と互換性を確保するために慎重にキュレーションされています。

## コレクションタイプ

- **オリジナルコレクション**：300の無料アバターを含む100Avatarsシリーズ（R1、R2、R3）
- **コミュニティコレクション**：オープンライセンスを使用するさまざまなクリエイターからのキュレーションされたコレクション

## ライセンス

ほとんどのコレクションは**CC0（パブリックドメイン）**ライセンスを使用しており、帰属表示なしで自由に使用できます。一部のコレクションは**CC-BY**ライセンスを使用しており、帰属表示が必要ですが、商用利用も可能です。特定のライセンス情報については、各コレクションの詳細を確認してください。
